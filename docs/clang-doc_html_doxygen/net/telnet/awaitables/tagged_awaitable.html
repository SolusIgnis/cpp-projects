class tagged_awaitable

Defined at line 37 of file net.telnet-awaitables.cppm

brief

Wrapper for an awaitable with a semantic tag for type safety.

remark

Provides implicit conversion to/from the underlying awaitable and supports direct `co_await`.

see

`tags` namespace for semantic tag types, `:protocol_fsm`, `:internal`
